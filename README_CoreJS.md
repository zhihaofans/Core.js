## AppKernel （应用的核心）

### `.START_TIME`：应用启动时间戳
### `.MOD_DIR`：指定Mod目录
### `.DEBUG`：是否为JSBox的调试模式
### `.DATA_DIR`：SHARED、ICLOUD、LOCAL目录的路径
### `.DEFAULE_SQLITE_FILE`：给Mod用的统一SQLite文件路径
### `.ModLoader`：应用核心初始化自动加载Mod加载器
